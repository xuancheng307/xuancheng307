ZK 研究與技術寫作

THESIS碩士論文 v4 — Polynomial Approximation of Transcendental Functions for ZK Financial Compliance

78 頁正式論文｜Halo2 PSE v0.3.0 / KZG/SHPLONK / BN254｜13 functions × 637 (pp,d) full ablation

架構設計（Chapter 3）

  • Four-layer architecture：Input Layer / Coefficient Layer / Precision Layer / Circuit Layer — 分離資料攝取、多項式係數、固定點精度管理、算術電路構造
  • 核心技術貢獻：Deferred Truncation with BN254 Headroom Analysis（§3.3.2）— 經驗 boundary「if (d+1) × pp < 254, else per-step」，讓 13 函數在 pp=16 全達 ULP=1

Protocol (Chapter 4 — Scenario A: Black-Scholes Delta-bounded Compliance)

  • Four-layer trust model：genesis anchor / exchange authentication / market data commitment / ZK computation / random audit
  • Dual-proof architecture：Proof #1 (compliance: 投資組合 Delta ≤ threshold) + Proof #2 (state-transition: state root RT 從 RT-1 正確衍生)
  • Two Merkle commitments：state root RT 與 market data root RM 分離

實驗結果

  • Polynomial evaluation 比 lookup 快 26.3×，lookup 在 pp ≥ 24 不可行
  • Minimax 比 Taylor 準 2× to 4,000×（依函數而異）
  • Merton scenario：6,944-byte proof + ~23ms verification + 28.6–29.0 effective bits
  • 首個 end-to-end ZK transcendental-function evaluation — Black-Scholes Delta hedging (pp=24) + Merton credit default (pp=32)
原創在數值分析層（Remez minimax + Horner + deferred truncation），ZK 系統用 stock Halo2/KZG — 不假裝 novel cryptography
