Research

Master's Thesis — Polynomial Approximation of Transcendental Functions for ZK Financial Compliance↗ case study
2024 — 2025-12
78 頁論文｜Halo2 PSE v0.3.0 / KZG/SHPLONK / BN254｜13 functions × 637 (pp, degree) full ablation grid

架構（Chapter 3）

  • Four-layer architecture：Input Layer / Coefficient Layer / Precision Layer / Circuit Layer
  • 核心技術貢獻 §3.3.2：Deferred truncation with BN254 headroom analysis，正式 boundary equation：if (d+1) × pp < 254 then deferred else per-step；讓 13 函數在 pp=16 全達 ULP=1

Protocol — Scenario A: Black-Scholes Delta-bounded Compliance

  • Four-layer trust model：genesis anchor / exchange authentication / market data commitment / ZK computation / random audit
  • Dual-proof architecture：Proof #1（compliance: 投資組合 Delta ≤ threshold）+ Proof #2（state-transition: state root RT 從 RT-1 正確衍生）
  • Two Merkle commitments：state root RT 與 market data root RM 分離

實驗結果

  • Polynomial evaluation 比 lookup table 快 26.3 倍，lookup 在 pp ≥ 24 不可行
  • Minimax 比 Taylor 展開準確 2 倍至 4,000 倍（依函數而異）
  • Merton scenario：6,944-byte proof 在 23 ms 內驗證，達 28.6–29.0 effective bits 精度
  • 首個 end-to-end ZK transcendental-function evaluation — Black-Scholes Delta hedging (pp=24) + Merton credit default (pp=32)

原創在數值分析層（Remez minimax + Horner + deferred truncation），ZK 系統使用 stock Halo2/KZG。
